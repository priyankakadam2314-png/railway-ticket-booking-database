# 🚆 Railway Ticket Booking Database (UTS System)

## 📌 Overview
SQL-based Railway Ticket Booking System inspired by Indian Railways UTS.  
Manages users, trains, routes, bookings, payments, and wallet system while demonstrating database design and analytical SQL queries.

## 🛠 Tools
- MySQL  
- SQL (DDL, DML, Joins, Aggregations)  

## 🗄 Features
- User & wallet management  
- Train & route management  
- Ticket booking & payments  
- Season tickets & QR booking  
- Automated ticket expiry (MySQL Events)  

## 📊 Key Insights
- Identified top booking users  
- Calculated total revenue  
- Found low-performing routes  
- Automated ticket expiry system  

## 📂 Files
- `setup.sql` – Table creation  
- `sample_data.sql` – Sample data  
- `sql_queries.sql` – Analysis queries

- Subqueries  
- Analytical SQL
